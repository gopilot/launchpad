Text Editors

  • Sublime Text - By far and away the most popular text editor for development
  • TextMate - Somewhat popular text editor for Mac
  • TextWrangler - Capable text editor for Windows and Mac
  • XCode - The only tool for iOS development
  • Eclipse - The most common tool for Android development

Git

Git is a version control tool that lets you save your code and revert mistakes if something goes wrong. Github is a hugely popular website that uses Git to allow people to collaborate on projects.

Hosting Providers

  • Heroku - Free, farily simple hosting with command-line deployment
  • Digital Ocean - Inexpensive hosting provider that gives you your own personal server
  • Github Pages - Free hosting for static websites, provided by Github
